# Django backend

## Installation

### Install virtualenv
```
$ pip install virtualenv
```

### Create virtual environment
```
$ virtualenv .venv
```

### Activate virtual environment
```
$ . .venv/bin/activate
```

### Install requirements
```
$ pip install -r requirements.txt
```

### Run migrations
```
$ ./manage.py migrate
```

### Run server
```
$ ./manage.py runserver
...
Django version 3.0.5, using settings 'model_garden.settings'

```
